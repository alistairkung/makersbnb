# User Stories

```
As a user so that I can view a property
I want to be able to see a list of available properties.

As a user so that I can stay in my desired location
I want to be able to search by location

As a user so that I can stay on a specific date
I want to be able to book my stay for a specific date(s)

As a user so that I can book my desired property
I want to be able to sign up for an account

As a user so that I can see if a property is available
I want to be able to see the available dates

As a user so that I can stay at a property
I want to be able to confirm my booking
```

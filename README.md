# User Stories

```
As a user so that I can view a property
I want to be able to see a list of available properties.

As a user so that I can stay in my desired location
I want to be able to search by location

As a user so that I can see if a property is available
I want to be able to see the available dates

As a user so that I can book my desired property
I want to be able to sign up for an account

As a user so that I can stay on a specific date
I want to be able to book my stay for a specific date(s)

As a user so that I can stay at a property
I want to be able to confirm my booking

As a user so that I can hire a space
I want to receive confirmation from the owner of a property

As an owner of a property
So I can make money from an empty property
I want to be able to list my property 

As an owner
So that I may attract potential holiday goers
I want to be able to provide a description, name and price

As an owner
So that my property is not double booked
I want to be able to provide a range of dates of availability 

As an owner
So that I can prevent damage to my property
I want to be able to reject troublesome customers

As an owner of many properties
So I can make even more money from my empty properties
I want to be able to list multiple properties



```
